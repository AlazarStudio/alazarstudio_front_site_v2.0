# React Project Admin

Универсальная админ панель на основе React, созданная для управления контентом различных проектов.

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

3. Настройте переменные окружения в `.env`:
```
VITE_API_URL=http://localhost:5000/api
```

## Запуск

### Режим разработки
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для продакшена
```bash
npm run build
```

### Просмотр собранного проекта
```bash
npm run preview
```

## Структура проекта

```
react-project-admin/
├── src/
│   ├── app/
│   │   └── admin/          # Вся админ панель
│   │       ├── layout.jsx  # Основной layout
│   │       ├── page.jsx    # Главная страница дашборда
│   │       ├── components/ # Компоненты админ панели
│   │       ├── places/     # Управление местами
│   │       ├── routes/     # Управление маршрутами
│   │       ├── news/       # Управление новостями
│   │       ├── services/   # Управление услугами
│   │       ├── reviews/    # Управление отзывами
│   │       ├── users/      # Управление пользователями
│   │       └── ...
│   ├── components/         # Общие компоненты
│   ├── contexts/           # React контексты
│   ├── lib/                # Утилиты и API
│   └── sections/           # Секции (используются в админ панели)
├── package.json
├── vite.config.js
└── index.html
```

## Особенности

- Полнофункциональная админ панель с управлением контентом
- Аутентификация и авторизация
- Управление пользователями с ролями (ADMIN, SUPERADMIN)
- CRUD операции для всех сущностей
- Загрузка и управление медиа файлами
- Редактор текста (Rich Text Editor)
- Интеграция с Яндекс.Картами
- Адаптивный дизайн

## Требования

- Node.js 18+
- npm или yarn
